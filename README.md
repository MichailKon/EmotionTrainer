# Russian
Для работы программы необходим Python 3.6.8
Все необходимые библиотеки устанавливаются с помощью requirements.txt
Для установки пропишите в терминале (командной строке) pip install -r requirements.txt

All necessary libraries are installed using requirements.txt

To run the program, run the MainCode.py file
